# 🧵 Vastra Vayana – Handloom Heritage Website

A modern, responsive web application showcasing traditional Indian handloom products and artisan craftsmanship.

## 🚀 Features
- Elegant UI inspired by Indian heritage
- Category-based product filtering
- Smooth animations and scroll effects
- Responsive design

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
vastra-vayana/
│── index.html
│── style.css
│── script.js
│── assets/

## 🔧 How to Run
Open index.html in your browser
